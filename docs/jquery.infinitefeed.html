jquery.infinitefeed.js

¶

Infinite Feed plugin

Infinite feed in a small abstraction for showing feeds. It comes with a infinite scrolling and loading management. underscore.js and [jscrollpane] are required. make sure to use proper css for the selector, e.g.:

 
             		           
                               
                            
                 ¶               
               

  

    
width: 100%;
    height: 200px;
    overflow: auto;

  


             		                            
             		           
                               
                            
                 ¶               
               
Infinite feed has this options:
             		                            
             		           
                               
                            
                 ¶               
               
    

  • header: is a string represnting html that will be showed on top of the feed. Defaults to empty string.
    • 

  • url: is a string of the feed url used in the json call. Required.
    • 

  • tmpl: is a function that gets a feed item and returns html. Required.
    • 

  • timer: how many seconds between fetching new content?
    • 

  • failure: how to detect failure in retrieving the feed. todo!.
    • 

  • firstTimeHook: function to call after the first loading
    • 

  • loadHook: function to call after every load. todo!.
    • 

  • firstLoading: function($selector) for the first loading
    • 

  • newLoading: html of the loading for fetching new items.
    • 

  • loadingClass: the selector of the loading elements. defaults to 'loading'.
    • 

  • timestamp: the attribute of the feed item that indicates its time.
    • 

  • id: the attribute of the feed item that indicates its id.
    • 

  • loadMoreRequest: the data we need to send the request in order to get items before the last one we have, message to send when we have reached the end.
    • 

  • getItems: returns the array of items from the data
    • 

  • footer: html in the footer of the pane..
    • 

  • notifyOnNewItems: gets called with storage object when there are new items
    • 

  • requestType: json,jsonp, etc
    •
